Cosa fa il workflow

Ogni venerdì a mezzogiorno n8n legge automaticamente i dati degli interventi tecnici dal foglio Google Sheets, li converte in CSV e li passa a Claude API con un prompt strutturato. Claude genera una dashboard HTML completa — KPI, grafici, tabelle riepilogative — in lingua italiana e con tema grafico coerente. Il file viene poi caricato su GitHub Pages tramite API, sovrascrivendo sempre lo stesso URL. Gmail invia la notifica con il link diretto. Zero configurazione settimanale, zero intervento umano.
